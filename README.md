# covid-iran
Reverse-engineered interesting data links from "Mask" app. (official Iranian government covid app)   
### Daily total stats: infected.json   
https://cdn2.covidapp.ir/data/infected.json   
### Type of available maps: maps.json
https://cdn2.covidapp.ir/map/maps.json   
### Map data: [id].[version].csv (including svg and stats)
https://cdn.covidapp.ir/map/shahrestan_label.400088.csv   
https://cdn.covidapp.ir/map/plot_shahrestan_label.400086.csv   
### Remote config: config.json
https://cdn.covidapp.ir/data/config.json   
### Version info: version.json
https://cdn.covidapp.ir/data/version.json
